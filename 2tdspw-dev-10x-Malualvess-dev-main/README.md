```

```

## Parte 1 — Criação do seu Prompt

Seu objetivo agora é:

1. Ler atentamente o arquivo `META-PROMPT.md`.
2. Preencher todas as informações solicitadas com seus próprios dados.
3. Criar um novo arquivo chamado:

```
PROMPT.md
```

4. Copiar o conteúdo do template para `PROMPT.md`.
5. Substituir todos os placeholders pelas suas informações reais.

---

### Regras

- Não deixe nenhum campo em branco.
- Não mantenha placeholders.
- Use apenas informações verdadeiras.
- Seja específico.
- Não invente experiências.
- Não altere a estrutura original do template.

---

### Resultado Esperado

Ao final desta etapa, o repositório deve conter:

```
META-PROMPT.md
PROMPT.md
```

O arquivo `PROMPT.md` deve conter a versão personalizada completa do template.

Faça o commit com uma mensagem clara e realize o push para o GitHub.

# Regras Finais e Critérios de Avaliação

Leia com atenção.

Todas as regras descritas neste README e comunicadas em aula **devem ser seguidas integralmente**.

O descumprimento de qualquer regra implica:

- Perda do direito à pontuação extra.
- Desconsideração da atividade para fins de bônus.

---

## Critérios para Ter Direito à Pontuação

Para estar apto a receber pontuação, o aluno deverá obrigatoriamente:

1. Ter seguido todas as instruções da atividade.
2. Ter realizado todas as etapas corretamente.
3. Ter feito os commits correspondentes a cada parte.
4. Ter submetido todos os arquivos solicitados.
5. Ter o conteúdo aprovado por uma IA orientada e controlada pelo professor.

A IA será utilizada como ferramenta auxiliar de análise, considerando:

- Coerência narrativa.
- Clareza de posicionamento.
- Evolução entre versões.
- Autenticidade.
- Consistência entre os textos.

A decisão final será sempre do professor.

---

## Pontuação

Alunos que cumprirem todos os critérios e tiverem seus materiais aprovados receberão:

**+2 pontos extras na CP1.**

---

## Bônus

A IA, sob supervisão direta do professor, irá selecionar o melhor “candidato” da turma com base no conjunto completo dos textos submetidos.

O aluno selecionado receberá:

**+5 pontos extras na CP1.**
